# Gantt Manager App

Diagrama Gantt y Kanban con drag & drop, prioridades, fechas límite y persistencia en el navegador.

## Características
- Vista Gantt interactiva con línea de "Hoy".
- Tablero Kanban para gestión de estados.
- Persistencia en `localStorage`.
- Filtros por estado, prioridad y búsqueda en tiempo real.
- Estadísticas de progreso.

## Instalación
```bash
npm install
```

## Uso
```bash
npm run dev
```
