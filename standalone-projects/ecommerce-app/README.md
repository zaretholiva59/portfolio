# Ecommerce App

Catálogo, carrito, wishlist, búsqueda y flujo de pago simulado con persistencia en el navegador.

## Características
- Catálogo de productos frescos con filtros por categoría.
- Carrito de compras funcional.
- Lista de deseos (Wishlist) persistente.
- Búsqueda en tiempo real y ordenación.
- Proceso de pago (Checkout) en 3 pasos.
- Persistencia en `localStorage`.

## Instalación
```bash
npm install
```

## Uso
```bash
npm run dev
```
