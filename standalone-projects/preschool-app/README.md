# Preschool App

Actividades, galería, calendario y anuncios para gestión de aulas preescolares.

## Características
- Checklist interactivo de actividades diarias.
- Galería de trabajos de estudiantes con vista previa.
- Calendario mensual de eventos.
- Sección de anuncios y noticias importantes.
- Diseño vibrante y amigable para niños.
- Persistencia en `localStorage`.

## Instalación
```bash
npm install
```

## Uso
```bash
npm run dev
```
